
# 通信
异构芯片间，通信层通过RDMA协议结合厂商通信库适配层完成。 
1. 厂商实现的专用高性能通信库，都对自家硬件适配度更高、性能更强。
2. 对于依赖RoCE协议，或者IB出网卡后网络协议相同的机器，实现通信协议，保障了跨品牌芯片通信和性能。
3. 芯片厂商不需要专门适配XCCL，只需要实现自身专用的集合通信库即可。
